## Description
### The purpose of this project is to demonstrate how one can use routes and database to communicate information about products and display said information. For this project we crated fake inventory and displayed how the tables of data (models in our case) are linked and how they can opperate with queries using api calls. 

## Setup
### make sure you're in the directory where you saved file then from command line 
### mysql -u root -p 
### source db/schema.sql
### npm run seed
